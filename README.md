# ZombieOutbreak

# Description
A survivor wakes up in a hospital to find a city in ruins and infested with zombies.
Run, kill and survive in a zombie outbreak.

# Controlls
Move using W/S/A/D
Aim using mouse
Fire Left click or Space Bar
